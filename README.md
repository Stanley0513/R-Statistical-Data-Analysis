# 📊 統計資料分析專案集

收錄了基於 **R 語言** 進行的統計資料分析專案，專案完整涵蓋了從探索性資料分析、統計假設檢定，到進階統計建模的完整流程，透過嚴謹的統計方法，從真實數據中獲得具決策價值之結果。

## 📁 專案架構 (Project Structure)

### 1. 📂 期中專案： 食物配方對凝血時間之影響
* **核心目標：** 針對資料進行初步的資料清洗與探索性資料分析。
* **實作技術：**
  * 敘述性統計與基礎資料前處理。
  * 運用 R 語言進行統計圖表視覺化。
  * 基礎統計假設檢定。
* **檔案：** `New.Rmd`, `黃柏翰張智賢.pdf`

### 2. 📂 期末專案： 年收入和風險態度對壽險購買金額的影響分析
* **核心目標：** 針對資料進行深度的統計建模與變數關聯性分析。
* **實作技術：**
  * 變數篩選與多重共線性檢驗。
  * 建立統計迴歸模型或變異數分析。
  * 模型殘差檢驗與配適度評估。
* **檔案：** `期末.Rmd`, `第十三組期末報告.pdf`

## 🛠️ 開發環境與技術
* **分析語言：** R
* **核心統計概念：** 探索性資料分析 (EDA)、假設檢定 (T-test/ANOVA)、迴歸分析 (Regression)、資料視覺化
* **開發工具：** RStudio, R Markdown
